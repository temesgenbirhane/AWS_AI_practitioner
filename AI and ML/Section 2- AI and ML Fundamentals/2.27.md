# What is a Model? (in general terms)

In general, a model is an informative representation of an object, person, or system.

Models can be:
- Concrete — have a physical form (e.g., a design for a vehicle, a person posing for a painting).
- Abstract — expressed as behavioral patterns (e.g., mathematical equations, computer code, written words).
  
What is Machine Learning Modeling (ML Model)?

An ML model is a function that takes in data and performs an ML algorithm to produce a prediction.
The ML model is trained on labeled data using a learning algorithm.
Once trained and tuned (optimized), the model can be deployed to make predictions on unlabeled data.
Do not confuse the training model (which learns) with the trained model (which predicts).

    ML Model Workflow

1) Training Phase

    -Input: Labeled Data<br>
    -Process: Learning Algorithm (training model)<br>
    -Optimization step: Hyper-tuning

2) Deployment Phase
   
    -The trained model is Deployed to handle new data.<br>
    -Input: Unlabeled Data<br>
    -Output: Prediction
    - Step: Inference (model making real-world predictions).

Summary

    -Training Model → learns from data.
    -Trained Model → deployed for prediction.
    -Hyper-tuning improves accuracy before deployment.
