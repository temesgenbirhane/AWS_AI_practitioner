# What is Jupyter Notebook?
Jupyter Notebook is a web-based application for creating documents that combine:

    - Live code
    - Narrative text
    - Equations
    - Visualizations
 
It’s lightweight and easy to use, but has a basic UI and limited workspace organization.

Originally developed as part of IPython, Jupyter Notebooks evolved into a standalone tool.

🔁 Over time, Jupyter Notebook was integrated into a more advanced interface called JupyterLab.

📎 The old interface is now referred to as Jupyter Classic Notebook.

## What is JupyterLab?
JupyterLab is the next-generation web-based user interface for Jupyter. It is the modern, more powerful IDE‑style environment for working with notebooks.

It includes:

    - Notebook
    - Terminal
    - Text Editor
    - File Browser
    - Rich Outputs
✅ Combines flexibility and power in one interface.
🔄 JupyterLab will eventually replace Jupyter Notebook.

## What is JupyterHub?
JupyterHub is a server-based platform that runs JupyterLab for multiple users.
JupyterHub is not an interface.
It’s a server system that lets many users run their own Jupyter Notebook or JupyterLab sessions on shared infrastructure.

Ideal for:

        - A class of students
        - Corporate data science teams
        - Scientific research groups
It manages authentication, spawns individual notebook instances, and controls access.
