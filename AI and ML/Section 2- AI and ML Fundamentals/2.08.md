# Divisions of Machine Learning

For Machine Learning, we can divide approaches into several broad categories.

## Classical Machine Learning

- Usage: For structured data with clear features.
- Examples: Supervised learning (regression, classification), unsupervised learning (clustering, dimensionality reduction).

## Reinforcement Learning

- Usage: When models need to learn optimal actions in an environment with no direct supervision.
- Applications: Real-time decision-making, game AI, robotics, and self-driving cars.

## Ensemble Methods

- Usage: Improve prediction quality by combining multiple models.
- Techniques:
	- Bagging (e.g., Random Forest)
	- Boosting (e.g., XGBoost)
	- Stacking

## Neural Networks & Deep Learning

- Usage: For unstructured data (images, text, audio) or when features are complex.
- Techniques:
	- Convolutional Neural Networks (CNNs): image and video processing.
	- Recurrent Neural Networks (RNNs): time-series or sequential data.
	- Generative Adversarial Networks (GANs): data generation tasks.
	- Multilayer Perceptron (MLP): basic feedforward networks.
	- Autoencoders: dimensionality reduction and anomaly detection.

## Quick comparison

| Category | Typical data | When to use |
|---|---:|---|
| Classical ML | Structured tabular data | When features are clear and dataset is small-to-medium |
| Deep Learning | Images, audio, text, complex features | When feature engineering is hard or large datasets exist |
| Reinforcement Learning | Sequential decision environments | When learning via trial-and-error is required |
| Ensembles | Structured data with noisy signals | When boosting accuracy and robustness matters |
