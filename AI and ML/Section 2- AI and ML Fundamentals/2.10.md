# Supervised Learning Models

## Classification

- Definition: The process of finding a function to divide a dataset into classes or categories.
- Example: Predicting whether tomorrow's temperature will be "hot" or "cold."

### Common Classification Algorithms

- Logistic Regression
- K-Nearest Neighbors (KNN)
- Support Vector Machines (SVM) / Kernel SVM
- Naïve Bayes
- Decision Tree
- Random Forest

## Regression

- Definition: The process of finding a function to predict a continuous variable.
- Example: Predicting tomorrow’s temperature in degrees.

### Common Regression Algorithms

- Simple Linear Regression
- Multiple Linear Regression
- Polynomial Regression
- Support Vector Regression (SVR)
- Decision Tree Regression
- Random Forest Regression

## Quick notes

- Choose classification when outputs are discrete categories; choose regression when outputs are continuous values.
- Tree-based ensembles (Random Forest, Gradient Boosting) often work well with mixed feature types and require less feature scaling.
- Linear and kernel methods can be effective on smaller, well-structured datasets.
