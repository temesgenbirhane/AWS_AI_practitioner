# Supervised vs Unsupervised Learning

## Supervised Learning (SL)

- Definition: A machine learning task that requires labeled training data.
- How it works: The model is trained on input–output pairs to learn the relationship and make predictions on new data.
- Analogy: “Show me how to do it, and then I can do it on my own.”

### Key Models

- Classification — Predicts discrete categories (e.g., spam or not spam).
- Regression — Predicts continuous values (e.g., house prices).

## Unsupervised Learning (UL)

- Definition: A machine learning task that works with unlabeled data to find patterns or hidden structures.
- How it works: The algorithm groups or organizes data based on similarities without predefined labels.
- Analogy: “I am an independent worker; I can figure this out on my own.”

### Key Models

- Clustering — Groups similar data points together (e.g., customer segmentation).
- Association — Finds rules or relationships (e.g., items frequently bought together).
- Dimensionality reduction — Simplifies data by reducing the number of variables.
## Reinforcement Learning(RI)
- There is no data, there is an environment and an ML model generates data any many attempt to reach a goal decisions-driven

### Examples
- Game AI, Learning Tasks, Robot Navigation

## Comparison Notes

- Supervised learning is generally more accurate but requires more labeled data and preparation.
- Unsupervised learning requires less upfront labeling but needs human validation and interpretation of results.

## Quick decision guide

| Goal | Prefer | Why |
|---|---|---|
| Predict known labels | Supervised | Uses labeled examples to learn direct mappings |
| Discover structure | Unsupervised | Finds clusters, associations, or lower-dimensional representations |
